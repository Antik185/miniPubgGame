# Импорт персонажа и оружия

## Персонаж

Предпочтительный формат — один файл `.glb`.

Требования:

- персонаж в A-pose или T-pose;
- humanoid-скелет, желательно Mixamo;
- кости `Hips`, `Spine`, `Head`, `RightHand`, `LeftHand`, руки и ноги;
- Skin Weight уже настроен;
- корень модели расположен между ступнями на уровне пола;
- рост после импорта около 1,7–1,8 м;
- текстуры встроены в GLB либо лежат рядом отдельными PNG/JPG;
- Base Color, Normal и Roughness без запечённого студийного света.

Оптимальный путь: Blender → Mixamo-compatible rig → экспорт glTF 2.0 (`.glb`).

`Black_Man_FBX.FBX` имеет формат FBX 6.1 и старый 3ds Max Biped-скелет.
Он сконвертирован в `Black_Man.glb`; основные кости вручную сопоставлены с
Mixamo и получают ретаргетированные idle/walk/run-анимации. Исходных файлов
текстур внутри набора нет, поэтому одежда и кожа восстановлены материалами.

`50centbloodonthesand_-_50cent.glb` содержит единый Skin, 119 костей,
встроенные PNG-текстуры, пальцы, лицевой риг и отдельные `GunBone` для рук.
Его humanoid-кости сопоставлены с игровой системой, а locomotion-анимации
ретаргетируются из общей библиотеки.

## Оружие

Оружию humanoid-скелет не требуется. Предпочтителен `.glb` со следующими
именованными точками (`Empty`/`Object3D`):

- `Grip_R` — центр рукояти для правой ладони;
- `Grip_L` — место левой руки на цевье;
- `Muzzle` — конец ствола;
- `Magazine` — отдельный магазин для перезарядки;
- `Eject` — точка выброса гильзы, если нужна.

Ствол в локальных координатах проекта направлен вдоль `-Z`. Анимации ходьбы,
прицеливания и перезарядки принадлежат персонажу. Скелет оружия нужен только
для подвижного затвора, барабана или других механических деталей.
