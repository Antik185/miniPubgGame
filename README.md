# Батл-рояль 3D (браузер)

Мини-«фортнайт» без стройки: 3D (three.js), вид от третьего лица через плечо, сужающаяся зона,
лут, хотбар на 5 слотов, боты-противники и эмоции. Офлайн (боты); онлайн — в планах.

## Запуск

Самый простой способ: дважды кликнуть `START_GAME.cmd`. Скрипт сам запустит
локальный сервер в фоне и откроет игру по адресу http://127.0.0.1:4173/.

Важно: напрямую открывать `index.html` нельзя — браузер блокирует ES-модули,
когда страница запущена через `file://`.

Ручной запуск из PowerShell:

```
python -m http.server 4173 --bind 127.0.0.1
```

Затем открыть http://127.0.0.1:4173/. Модели и three.js пока загружаются с CDN,
поэтому при первом запуске нужен интернет.

## Управление

| Клавиша | Действие |
|---|---|
| WASD | движение |
| Shift | включить / выключить бег (расходует стамину) |
| R | перезарядить оружие |
| Space | прыжок / залезть на камень |
| Ctrl (или C) | присесть; в беге — подкат |
| ЛКМ | огонь / использовать предмет |
| ПКМ | прицел (у снайперки — скоуп от первого лица) |
| E | подобрать лут |
| 1–5 / колесо | слоты хотбара |
| B | круг эмоций (выбор цифрой или кликом) |

Броня: синие бочки (малая +25, большая +50, максимум 100), пули сначала снимают броню.
Аптечка +50 HP. Расходники занимают слот и используются по ЛКМ.

## Структура

- `js/main.js` — игровой цикл, камера, стрельба, HUD, миникарта
- `js/character.js` — персонаж: реестр `CHARACTERS` (модель + карта анимаций + эмоции)
- `js/bots.js` — ИИ ботов (блуждание, стрельба, урон)
- `js/world.js` — остров, индустриальная локация, PBR-материалы и GLB-окружение
- `js/weapons.js` — характеристики и модели оружия
- `js/zone.js` — фазы шторма
- `js/loot.js` — спавн и подбор лута
- `js/audio.js` — процедурные звуки (WebAudio, без файлов)
- `2d/` — старая 2D-версия (самостоятельная, открой `2d/index.html`)

## Как добавить своего персонажа

Персонажи — mixamo-риги, все анимации (ходьба, бег, танец) берутся из общей
библиотеки (`ANIM_SOURCES` в `js/character.js`) и подходят любому mixamo-персонажу.
Чтобы добавить нового: экспортируй модель с Mixamo (glb) и допиши запись
в `CHARACTERS` (url + rotY) — эмоции и оружие заработают автоматически.
В обычной стойке оружие привязано к ладони и повторяет направление пальцев:
опущенная рука опускает ствол. При прицеливании поза плавно переходит в
двухручный хват с IK для рукояти и цевья.

## Сторонние ассеты

- Kenney City Kit (Industrial), CC0 — здания, дымовые трубы и промышленные ёмкости
- Kenney Nature Kit, CC0 — хвойные деревья и камни

Оригинальные лицензии лежат рядом с наборами в `assets/vendor/`.

## Отладка

В консоли доступен `window.__game` (scene, player, bots, zone, `step(n)`, `shoot()` и т.д.).
Для быстрой проверки позы можно открыть `?weapon=rifle&aim=1` или
`?weapon=sniper&aim=1&pose=crouch`.

## План на онлайн

Состояние матча (позиции, HP, зона) обновляется в `tick()` — для мультиплеера
его можно перенести на сервер (Node.js + WebSocket), клиенту оставить ввод и рендер.
